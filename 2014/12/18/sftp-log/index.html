CentOS下配置SFTP操作日志

SFTP操作日志的重要性，不言而喻，之前也找过资料，一直没找着…… 今天同事又提过这事，伤心的我，又去找了找，果然是老天不负我，竟然给我找着了，感激流涕中…… 不敢私藏，赶紧分享给大伙！同时感谢Caesar的技术支持！

1、修改SSH配置：

1
2
3

# vim /etc/ssh/sshd_config
Subsystem       sftp    /usr/libexec/openssh/sftp-server -l INFO -f local5
LogLevel INFO

2、syslog.conf末尾增加一行：

1
2

# vim /etc/syslog.conf
auth,authpriv.*,local5.* /var/log/sftp.log

3、重新加载服务：

1
2

# /etc/init.d/syslog reload
# /etc/init.d/sshd reload

4、查看日志：

1
2
3
4

# tail -f /var/log/sftp.log
Dec 18 16:41:56 localhost sftp-server[26998]: session opened for local user root from [192.168.1.9]
Dec 18 16:42:11 localhost sftp-server[26998]: opendir "/home/abc"
Dec 18 16:42:11 localhost sftp-server[26998]: closedir "/home/abc"

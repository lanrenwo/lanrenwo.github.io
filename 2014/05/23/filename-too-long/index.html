wget报File name too long

  • wget
  • weed-fs

最近在研究Weed-FS，没想到，下载这一步，就遇到了问题，提示File name too long！

# wget http://dl.bintray.com/chrislusf/Weed-FS/weed_0.58_linux_amd64.tar.gz
--2014-03-29 10:15:02-- http://dl.bintray.com/chrislusf/Weed-FS/weed_0.58_linux_amd64.tar.gz
Resolving dl.bintray.com... 108.168.194.92, 108.168.194.91
Connecting to dl.bintray.com|108.168.194.92|:80... connected.
HTTP request sent, awaiting response... 302
....
...
... 4M5VTQ' (File name too long)

赶紧Google一下，你就知道。 原来答案是如此的简单，真是孤陋寡闻，真得好好学习，天天向上。

wget -O weedfs.tar.gz http://dl.bintray.com/chrislusf/Weed-FS/weed_0.58_linux_amd64.tar.gz
